## Tips for Implementation

- Use `pygame.draw` for simple charts and graphs
- Implement `random.randint()` for price volatility
- Create sprite classes for stocks/traders
- Use `pygame.font` for retro-style text displays
- Add sound effects with `pygame.mixer` for trades
- Keep color palette limited (8-bit aesthetic)
- Use pixel fonts like "Press Start 2P" or "Consolas"
